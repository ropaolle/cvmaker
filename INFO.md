## Setup

```sh
pnpm create vite cvmaker --template react-ts
# Add Tailwind
pnpm install -D tailwindcss postcss autoprefixer
pnpm dlx tailwindcss init -p

```

```json
{
  "files.associations": {
    "*.css": "tailwindcss"
  }
}
```
